# 数据目录说明

`data/` 只保留三类内容：

- `reference/`：低频更新的参考快照，例如标的池清单。
- `samples/`：随仓库提交的最小可复现样例，供 README、测试和演示使用。
- `processed/`：下载、清洗和临时整理后的运行数据，默认不纳入版本控制。

## 提交边界

- 可以提交：小体积、明确用途、可复现文档或测试所必需的样例。
- 不应提交：大批量历史行情、一次性实验数据、临时导出 CSV。

## 默认样例

- `samples/1810_hk_daily.csv`
- `samples/1810_hk_15m.csv`

如果需要新增样例，请先说明它解决的文档或测试场景，避免仓库重新膨胀为数据工作目录。
