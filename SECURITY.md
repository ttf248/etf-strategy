# 安全策略

Strategy Studio 当前仍处在快速迭代阶段，但安全问题需要走单独通道处理。

## 适用范围

- API 鉴权或权限绕过
- 数据泄露
- 任意命令执行
- 依赖漏洞导致的高风险利用面
- 可能影响用户本机进程、数据库或文件系统的缺陷

## 反馈方式

请不要直接公开提交安全 Issue。请改用私下渠道发送最小复现信息：

- 问题描述
- 影响范围
- 复现步骤
- 可能的缓解方式

如果仓库未来接入正式邮箱或安全平台，应以该专用渠道为准；在此之前，请先通过维护者约定的私下沟通方式联系。

## 处理承诺

- 维护者确认收到后，会先判断是否可复现以及是否需要临时缓解。
- 高风险问题会优先修复，再补公开说明。
- 公开披露前，应给维护者留出合理修复窗口。
