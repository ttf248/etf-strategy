# 回测报告阅读指南

这份文档解决一个问题：打开报告之后，应该先看什么，图和数字分别代表什么。

## 建议阅读顺序

当前正式报告采用双层结构：

1. `第一层：先看结论`
2. `第二层：展开细节`

建议顺序是：

1. 先看 `先回答关键问题`
2. 再看 `一句话判断`
3. 如果值得继续，再看“参数是怎么选的”“关键结果对照”“执行口径和风控约束”“网格到底有没有帮忙”“左侧行情怎么处理”
4. 最后才看交易记录表

## 第一层到底在说什么

第一层只回答几类关键问题：

1. 这套策略现在能不能证明自己能赚钱
2. 它比“现金闲置不交易”到底好还是差
3. 它比“买入持有”到底好还是差
4. 这组参数是不是只在一小段样本里碰巧好看

如果你只是想判断“这套策略还值不值得继续研究”，通常到这里就够了。

## 第二层怎么看

第二层是解释层，核心分成三块：

### 1. 样本内寻参结果

这里回答：

- 参数为什么会入选
- 现在不是只看单窗最高分，而是会看多窗口稳健性
- 这组最优参数在旧样本里表现如何

### 2. 样本外验证

这里回答：

- 把样本内最优参数拿到新数据上，还灵不灵
- 是继续亏、转正，还是回到盈亏平衡附近

### 3. 交易记录

这里不是给你看结论的，而是给你追交易过程和盈亏归因的。

## 这些数字最容易看错

### 网格已实现收益

它只代表已经完成“买入 -> 卖出”闭环、真正落袋的利润。

它不等于整个账户总收益。

报告里新增了“网格到底有没有帮忙”对比表，建议和“现金闲置收益率”“买入持有收益率”一起看。

### 左侧处理口径

默认报告会同时展示：

- `hold`：持续下跌时不强制卖出，未平网格浮动盈亏留到账户权益里。
- `force_exit`：未平网格浮亏达到阈值后强制卖出，并停止继续开新网格。

这张表专门回答：如果遇到左侧单边下跌，网格是继续持有，还是先承认亏损退出。

### 执行口径

当前命令默认使用 `realistic` 口径，报告会展示手续费、滑点、最大仓位占用和停手事件。

如果你看到 `research` 口径，说明这份报告是在简化撮合假设下生成的，更适合做策略方向研究，不适合直接当作接近实盘的结果。

### 样本内/样本外收益率

这是整个账户口径的结果，包含：

- 未平仓网格仓位浮盈浮亏
- 已经落袋的网格利润
- 没有投入的现金余额

所以经常会出现：

- 网格已实现收益是正的
- 但总账户收益率仍然是负的

### 期末有效持仓成本

这个数不是实际下单时的成交价，而是：

- 把已经兑现的网格利润扣回去后
- 剩余未平网格仓位的等效成本

如果期末没有未平网格仓位，这个数会回到 `0`。

## 回测图怎么读

回测图通常分 4 层：

### 价格层

- 收盘价
- 持仓有效成本
- 买卖点散点

你主要看：

- 价格最后停在有效成本上方还是下方
- 网格买卖点密不密

### 资金层

- 当前投入资金
- 累计网格已实现收益

你主要看：

- 网格有没有持续兑现利润
- 仓位是不是越跌越重

### 收益层

- 组合收益率
- 回撤比例

你主要看：

- 总账户最终是不是回正
- 中间最差的时候回撤有多深

### 风控与成本层

- 仓位占用
- 仓位上限
- 累计手续费
- 累计滑点

你主要看：

- 网格是不是越跌越接近仓位上限
- 交易成本有没有因为频繁买卖持续抬高

## 热力图怎么读

热力图的含义是：

- 横轴：网格间距
- 纵轴：网格层数
- 颜色：综合评分高低

当前图里每个格子展示的是：

- 对应“间距 + 层数”组合下
- 在所有候选止盈比例里能拿到的最好稳健结果

所以它更适合回答：

- 高分区域大概集中在哪
- 最优点是不是稳定区域的一部分
- 网格层数继续加深还有没有明显收益

## 什么时候继续看交易记录

只有在下面这些情况，才建议往下翻交易记录：

- 你怀疑报告里的结论和图不一致
- 你想知道某次大回撤是怎么形成的
- 你想确认网格闭环到底发生了几次
- 你准备改策略逻辑，需要核对执行路径
