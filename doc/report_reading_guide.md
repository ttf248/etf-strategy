# 回测报告阅读指南

这份文档解决一个问题：打开报告之后，应该先看什么，图和数字分别代表什么。

## 建议阅读顺序

当前正式报告采用双层结构：

1. `第一层：先看结论`
2. `第二层：展开细节`

建议顺序是：

1. 先看 `老板一眼看懂版`
2. 再看 `怎么使用这份报告`
3. 如果值得继续，再看样本内和样本外的细节
4. 最后才看交易记录表

## 第一层到底在说什么

第一层只回答 3 个问题：

1. 样本内这套参数综合结果是赚钱还是亏钱
2. 当前最值得关注的参数组合是什么
3. 样本外有没有延续性

如果你只是想判断“这套策略还值不值得继续研究”，通常到这里就够了。

## 第二层怎么看

第二层是解释层，核心分成三块：

### 1. 样本内寻参结果

这里回答：

- 参数是怎么选出来的
- 这组最优参数在旧样本里表现如何
- 网格到底帮你赚到了什么、又没帮你解决什么

### 2. 样本外验证

这里回答：

- 把样本内最优参数拿到新数据上，还灵不灵
- 是继续亏、转正，还是回到盈亏平衡附近

### 3. 交易记录

这里不是给你看结论的，而是给你追交易过程和盈亏归因的。

## 这些数字最容易看错

### 网格已实现收益

它只代表已经完成“买入 -> 卖出”闭环、真正落袋的利润。

它不等于整个账户总收益。

### 样本内/样本外收益率

这是整个账户口径的结果，包含：

- 底仓浮盈浮亏
- 未平仓网格仓位浮盈浮亏
- 已经落袋的网格利润

所以经常会出现：

- 网格已实现收益是正的
- 但总账户收益率仍然是负的

### 期末有效持仓成本

这个数不是实际下单时的成交价，而是：

- 把已经兑现的网格利润扣回去后
- 剩余持仓的摊薄成本

它更适合用来判断“网格有没有把手里仓位成本压低”。

## 回测图怎么读

回测图通常分 3 层：

### 价格层

- 收盘价
- 持仓有效成本
- 买卖点散点

你主要看：

- 价格最后停在有效成本上方还是下方
- 网格买卖点密不密

### 资金层

- 当前投入资金
- 累计网格已实现收益

你主要看：

- 网格有没有持续兑现利润
- 仓位是不是越跌越重

### 收益层

- 组合收益率
- 回撤比例

你主要看：

- 总账户最终是不是回正
- 中间最差的时候回撤有多深

## 热力图怎么读

热力图的含义是：

- 横轴：网格间距
- 纵轴：网格层数
- 颜色：综合评分高低

当前图里每个格子展示的是：

- 对应“间距 + 层数”组合下
- 在所有候选止盈比例里能拿到的最好结果

所以它更适合回答：

- 高分区域大概集中在哪
- 最优点是不是稳定区域的一部分
- 网格层数继续加深还有没有明显收益

## 什么时候继续看交易记录

只有在下面这些情况，才建议往下翻交易记录：

- 你怀疑报告里的结论和图不一致
- 你想知道某次大回撤是怎么形成的
- 你想确认网格闭环到底发生了几次
- 你准备改策略逻辑，需要核对执行路径
